# Mini Crossword

A compact 5×5 crossword built as a word square. Solve the across and down clues to fill every cell.

## Gameplay
- Tap or click a cell to select it. The highlighted run shows the active clue.
- Type letters on a keyboard or tap the onscreen cell and use the software keyboard on mobile.
- Arrow keys move between cells. Press space or enter to toggle between across and down mode.
- The Reset button clears the grid so you can start over.

## Save data
This early prototype does not persist puzzle progress yet. Each session starts fresh.

## Difficulty
Designed as an introductory crossword that can be completed in a minute or two.
